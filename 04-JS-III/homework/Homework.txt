Resolución:

Arrays:
Los arrays son listas de elementos de cualquier tipo que se identifican por un índice. Pueden aceptar distintos tipos de datos y objetos y ser multidimensionales 
Así, si lista = ['a', 'b', 'c'] => lista[0] = 'a'
Existen funciones importantes para manejar aarreglos, como: 
* lista.length() => devuelve la cantidad de elementos dentro de un arreglo
* lista.push() => agrega un elemento al final del arreglo
* lista.pop() => elimina el último elemento del arreglo
* lista.unshift() => agrega un elemento al principio del arreglo
* lista.shift() => elimina el primer elemento del arreglo
