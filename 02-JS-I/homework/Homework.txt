Solicitado:

En un archivo de texto separado que debes crear, escribe explicaciones de los siguientes conceptos como si se lo estuvieras explicando a un niño de 12 años. Hacer esto te ayudará a descubrir rápidamente cualquier agujero en tu comprensión.
Variables
Strings
Funciones (argumentos, return)
Declaraciones if
Valores booleanos (true, false)

Respuesta:

a) Variables: son espacios de memoria que funcionan como contenedores de datos identificados con un nombre. 
Este nombre es único para cada variable y no pude ser una palabra reservada del lenguaje

b) Strings: Tipo de dato que se compone de caracteresuna cadena de uno o más caracteres alfanuméricos

c) Funciones (argumentos, return):
Secciones de código que se definen para ser reutilizados varias veces dentro del programa.
Las funciones pueden tener argumentos que son valores que se le pasan al ser invocadas. Estos argumentos son variables cuto scope es el de la función.
Las funciones pueden retornar un valor de cualquier tipo, si se usa la palabra reservada return.

d) Declaraciones if:
Sirven para controlar el flujo del código. Evalúan una condición y si es verdadera ejecutan el código entre llaves.
Puede usarse la sentencia else para ejecutar código cuando la condición es falsa.
Pueden anidarse declaraciones con el formato else if para evaluar distintas condiciones

e) Valores booleanos (true, false)
True y false son los valores que pueden tomar las variables de tipo Booleano, y también es lo que devuelven las condiciones al ser evaluadas.
Una condición devuelve false cuando es falsa y true si es verdadera.
